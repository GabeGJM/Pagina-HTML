Apresentando o Curso

Este curso é voltado para a introdução na programação web, onde iremos aprender os elementos básicos do HTML e como utilizalos na prática.

Ferramentas

A ferramenta que será utilizada é o Visual Studio Code, que é um editor de código-fonte desenvolvido pela Microsoft. O Visual Studio Code permite que o usiario instale plug-ins facilitar a edição de códigos. Nesse curso estaremos utilizando o Live Server para facilitar a vizualização do que estamos criando.

Instpetor de Elementos

O Inspetor de Elementos é uma ferramenta de desenvolvedor utilizada nos Navegadores Web. A ferramenta pode ser acessada(no chrome) pressionando a tecla F12, e lá é possivel visualizar como o site foi criado em HTML e formatado em CSS, para fins de conhecimento e estudo.

Estrutura Básica do HTML

Primeiramente o HTML necessita da criação de uma pasta dedicada para o projeto que será utilizada dentro do Visual Studio Code. Dentro do VS Code a pasta desejada será aberta e lá será criado um arquivo html, que tem como base tres partes: <html></html>, <head></head> e <body></body>

Falando sobre Tags

Algumas tags utilizadas no HTML são:

  • <i></i> - Utilizado para deixar o texto em itálico
  • <strong></strong> - Utilizado para deixar o texto em negrito
  • <u></u> - Utilizado para deixar o texto sublinhado

Atributos das Tags

Alguns atributos de tags utilizados são:

  • <input type="x" /> - Utilizado para criar um campo onde o usuário pode interagir
    • type="text" - Para digitar textos
    • type="number" - Para digitar números
    • type="color" - Para selecionar uma cor
  • id="x" - Utilizado para criar um identificador para o elemento
  • style="x" - Utilizado para formatar o elemento em comandos CSS
  • class="x" - Utilizado para criar uma classe para padronizar outros elementos em comandos CSS
  • <img src="x" /> - Para introduzir uma imagem

Textos

Existem várias tags HTMLs voltados para textos, algumas delas são:
  • <h1/>,<h2/>,<h3/>,<h4/>,<h5/>,<h6/> - Utilizados para denominar um título ou um subtítulo.
  • <p> - Para criar um parágrafo
  • <blockquote> - Para denominar uma citação
  • <mark> - Para marcar uma parte do texo
  • <sup> - Para elevar o texto

Listas Ordenadas e Não Ordenadas

As tags utilizadas para listas Ordenadas e Não Ordenadas são:

h3>
  • <ol> - Listas Ordenadas, que são as que possuem uma ordem
  • <ul> - Listas Não Ordenadas, que são as que não tem ordem determinada
  • <li> - Utilizado para denominar os pontos dentro das listas

Links

Links são utilizados para ligar uma página á outra, algumas tags que tem essa função são:

  • <a href="http://x.com">site</> - Utilizado para inserir um link á um elemento
    • Atributo target="_self - Para abrir o link na mesma guia
    • Atributo target="_blank - Para abrir o link em outra guia
